# Python-Data-Structures
В этом проекте я реализовала серию скриптов на Python для работы с различными структурами данных, файлами, словарями и строками. Каждый этап оформлен в отдельной директории внутри src

## Структура проекта

- **src/file_processing** — чтение и преобразование файлов (CSV → TSV).
- **src/stocks** — работа со словарями: поиск по ключу и значению, обработка нескольких запросов.
- **src/dictionaries** — преобразование списков в словари и сортировка.
- **src/sets** — операции с множествами для маркетинговых задач.
- **src/emails** — обработка email-адресов и генерация писем.
- **src/caesar** — реализация шифра Цезаря.

---

## 1. Работа с файлами (src/file_processing)

Я реализовала скрипт `read_and_write.py`, который корректно читает CSV-файл с вакансиями, заменяет разделители-запятые на табуляцию (учитывая, что данные могут содержать запятые внутри полей), и сохраняет результат в TSV-файл `ds.tsv`.

### Пример запуска
```bash
cd src/file_processing
python3 read_and_write.py
```
*Результат: файл ds.tsv*

## 2. Поиск по ключу и значению, обработка нескольких запросов (src/stocks)

Здесь я объединила логику поиска по названию компании и по тикеру, а также реализовала обработку строки с несколькими запросами через запятую. Скрипт `all_stocks.py` определяет, что введено: название компании или тикер, выводит цену акции или сообщение о неизвестном объекте. Скрипт не чувствителен к регистру и игнорирует лишние пробелы. Если аргументов нет, слишком много или есть пустые элементы (две запятые подряд), ничего не выводится.

### Примеры запуска
```bash
cd src/stocks
python3 all_stocks.py 'TSLA , aPPle     , Facebook'
```
*Результат:*
```
TSLA is a ticker symbol for Tesla
Apple stock price is 287.73
Facebook is an unknown company or an unknown ticker symbol
```
```bash
python3 all_stocks.py 'TSLA,, apple'
```
*Результат: (пустой вывод)*
```bash
python3 all_stocks.py TSLA AAPL
```
*Результат: (пустой вывод)*

## 3. Преобразование списков в словари и сортировка (src/dictionaries)

В этом разделе я реализовала два скрипта:
- `to_dictionary.py` — преобразует список кортежей (страна, число) в словарь, где ключом выступает число, а значением — страна. Выводится словарь в требуемом формате.
- `dict_sorter.py` — создаёт словарь (страна: число) и выводит названия стран сначала по убыванию числа, а при равенстве — по алфавиту.

### Примеры запуска
```bash
cd src/dictionaries
python3 to_dictionary.py
```
*Результат: словарь в требуемом формате*
```bash
python3 dict_sorter.py
```
*Результат: отсортированный список стран*

## 4. Операции с множествами для маркетинга (src/sets)

Я написала скрипт `marketing.py`, который решает три бизнес-задачи с помощью операций над множествами:
- поиск клиентов, не видевших промо-письмо;
- поиск участников события, не являющихся клиентами;
- поиск клиентов, не участвовавших в событии.
Скрипт принимает аргумент (`call_center`, `potential_clients`, `loyalty_program`) и возвращает соответствующий список email-адресов.

### Примеры запуска
```bash
cd src/sets
python3 marketing.py call_center
```
*Результат: список email для call-центра*
```bash
python3 marketing.py potential_clients
```
*Результат: список потенциальных клиентов*
```bash
python3 marketing.py loyalty_program
```
*Результат: список для программы лояльности*

## 5. Обработка email-адресов и генерация писем (src/emails)

Здесь реализовано два скрипта:
- `names_extractor.py` — извлекает имя и фамилию из email-адресов формата name.surname@corp.com, приводит их к нужному виду и сохраняет таблицу в TSV-файл.

*Результат: шаблон письма*

## 6. Шифр Цезаря (src/caesar)

Я реализовала скрипт `caesar.py`, который умеет кодировать и декодировать строки с заданным сдвигом. Скрипт поддерживает только латиницу: при встрече кириллицы выдаёт исключение. При неверном количестве аргументов также возникает ошибка.

### Примеры запуска

#### Кодирование команды:
```bash
python3 caesar.py encode 'ls -la /home/user' 10
```
*Результат: v{ -kv /voze~gyfe*
#### Декодирование:
```bash
python3 caesar.py decode 'v{ -kv /voze~gyfe' 10
```
*Результат: ls -la /home/user*
#### Ошибка при кириллице:
```bash
python3 caesar.py encode 'Привет, мир!' 3
```
*Результат: The script does not support your language yet.*
